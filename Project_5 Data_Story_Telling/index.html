Project Objective:

Understand the differences in voting patterns between the 2023 Turkish presidential elections and the 2024 local elections. Analyze factors that led to Erdogan's contrasting outcomes in these elections.

Stakeholders Requests:

  • How did voting patterns change between the two elections?
  • Were there specific regions where the shift in support was more pronounced?
  • Factors that led to the result?

Technology Stack:

  • JavaScript(D3.js), HTML, CSS
  • GIT
  • Jupyter Notebook, Python (Pandas & Numpy for data cleansing & analysis)

The charts show three key economic indicators for Turkey before the 2024 local elections.

  • A steady increase in the Consumer Price Index (CPI) indicates rising inflation, which may have affected voters' views on economic management.
  • The ongoing devaluation of the Turkish Lira against the US Dollar points to economic instability, potentially driving voters towards change.
  • Despite a modest increase in GDP per capita, its impact is subdued by the high inflation rates.

Results in the Metropolitan Cities

  • Data indicates a correlation between the CHP's victories in major cities and economic indicators like rising CPI and Lira devaluation.
  • In Istanbul and Ankara, the CHP's significant leads may reflect voter inclination towards parties promising economic reform.
  • The modest rise in GDP per capita does not appear to counteract the inflation's impact on electoral outcomes in metropolitan areas.

Project Description

Below is a series of steps outlining the process and techniques used in this project:

  • Initiated the data collection process by gathering information from various sources including Statista and specialized Turkish websites focusing on election results. This involved manually extracting data and ensuring its accuracy and relevance.
  • Transitioned to Jupyter Notebook to perform data cleaning and analysis using Python and Pandas. Tasks included detecting and handling duplicates, converting data to the Latin alphabet, renaming columns, and rectifying faulty data to create a comprehensive and usable DataFrame.
  • Converted the GeoJSON file to the Latin alphabet to ensure compatibility and uniformity.
  • Merged the GeoJSON data of cities with the election results of each respective year to create a cohesive dataset.
  • Utilized the D3 library to leverage the GeoJSON file for displaying the map and color-coded it based on the winning party.
  • Implemented interactive tooltips to enhance user experience and engagement with the map.
  • Developed three insightful charts using Statista data converted to a JSON file, with added tooltips and interactivity for enhanced data exploration.
  • Styled and provided additional context for the maps and charts, aiming to address stakeholders' inquiries with precision and clarity.
